# Anti-Cheat
DuckyAntiCheat is a lightweight but effective anti-cheat plugin built for Minecraft Spigot 1.21.10. It automatically detects and kicks players using common hacks — keeping your server fair and clean without banning them.

# Features
# Speed Check 
Detects players moving faster than normal

# Fly Check 
Prevents illegal flight movement

# NoFall Check 
Stops players from avoiding fall damage

# Scaffold / Tower Check 
Detects scaffold and tower building hacks

# Phase / Clip Check 
Detects players walking through blocks

# Automatic Kick System 
Instantly kicks cheaters instead of banning

# Custom Alerts
Sends messages to admins when cheats are detected

# Commands & Permissions
No commands needed! The plugin works automatically in the background.

# Permissions:
duckyanticheat.alerts → Receives cheat alerts in chat
